# Runbooks

Procédures opérationnelles pour la gestion du homelab.

## Alertes et runbooks associés

| Alerte | Runbook | Sévérité |
|--------|---------|----------|
| ServiceDown | [Service Down](./service-down) | Critical |
| DiskAlmostFull | [Disk Full](./disk-full) | Warning |
| HighCpuUsage | [High CPU](./high-cpu) | Warning |
| CertificateExpiring | [Certificate Expired](./certificate-expired) | Warning |

## Format des runbooks

Chaque runbook suit ce format :

1. **Symptômes** : Comment détecter le problème
2. **Impact** : Quels services sont affectés
3. **Diagnostic** : Commandes pour investiguer
4. **Résolution** : Étapes pour corriger
5. **Prévention** : Comment éviter à l'avenir
