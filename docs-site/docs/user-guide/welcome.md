# Bienvenue

Bienvenue sur la plateforme Smadja. Ce guide s’adresse aux **utilisateurs** des services (famille, proches, clients).

## Ce que vous trouverez ici

- **Services disponibles** — Liste des applications, à quoi elles servent et comment y accéder
- **Recommandations** — Bonnes pratiques pour bien utiliser les services
- **Limites** — Quotas (stockage, etc.) et ce qu’il faut savoir

## En cas de problème

- **Un service ne répond pas ?** Consultez la [page de statut](https://status.smadja.dev) pour voir si un incident est en cours.
- **Bug ou idée d’amélioration ?** Utilisez le [portail de feedback](https://feedback.smadja.dev) pour nous le signaler.

## Accès rapides

| Lien | Usage |
|------|--------|
| [home.smadja.dev](https://home.smadja.dev) | Page d’accueil avec tous les services |
| [auth.smadja.dev](https://auth.smadja.dev) | Connexion unique (compte utilisateur) |
| [status.smadja.dev](https://status.smadja.dev) | État des services |
| [feedback.smadja.dev](https://feedback.smadja.dev) | Signaler un bug ou une idée |
